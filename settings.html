⚙️ Tab Tamer Settings

Configure your automatic Notion integration

1 Create Notion Integration

1. Go to notion.so/my-integrations

2. Click "New integration"

3. Give it a name like "Tab Tamer"

4. Select your workspace

5. Click "Submit" and copy the "Internal Integration Token"

2 Select Parent Page

1. In Notion, navigate to or create a page where you want to store your Tab Tamer sessions

2. Copy the page ID from the URL (the 32-character string after the last slash)

3. Tab Tamer will automatically create a "Tab Tamer Sessions" database on this page with these properties:

  • Session Title (Title)
  • Summary (Rich Text)
  • Date (Date)
  • Tab Count (Number)
  • Category Count (Number)

Note: If the database already exists, Tab Tamer will use it automatically!

3 Configure Tab Tamer
Your Notion integration token (starts with "secret_")
The page ID where Tab Tamer will create/find the "Tab Tamer Sessions" database
4 Share Database with Integration

1. Go to your Notion database

2. Click "Share" in the top right

3. Click "Add people, emails, groups, or integrations"

4. Search for "Tab Tamer" and select your integration

5. Give it "Can edit" permissions
