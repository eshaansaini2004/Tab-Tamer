⚙️ Tab Tamer Settings

Configure your Notion integration

1 Create Notion Integration

1. Go to notion.so/my-integrations

2. Click "New integration"

3. Give it a name like "Tab Tamer"

4. Select your workspace

5. Click "Submit" and copy the "Internal Integration Token"

2 Create Notion Database

1. In Notion, create a new page

2. Add a database with these properties:

  • Title (Title)
  • Summary (Text)
  • Date (Date)
  • Tab Count (Number)
  • Cluster Count (Number)

3. Copy the database ID from the URL (the long string after the last slash)

3 Configure Tab Tamer
Your Notion integration token (starts with "secret_")
The database ID from your Notion database URL
4 Share Database with Integration

1. Go to your Notion database

2. Click "Share" in the top right

3. Click "Add people, emails, groups, or integrations"

4. Search for "Tab Tamer" and select your integration

5. Give it "Can edit" permissions
